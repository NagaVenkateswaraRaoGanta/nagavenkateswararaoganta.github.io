## Run Locally

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/venky-portfolio.git
```

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```

Open in browser:

```
http://localhost:8080
```
